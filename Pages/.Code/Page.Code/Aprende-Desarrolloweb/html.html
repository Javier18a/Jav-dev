Aprende HTML

Qué es Html ?

Html es el lenguaje de marcado de hipertexto, es decir, es el lenguaje utilizado para estructurar el contenido de una página, utilizamos html como la herramienta para escribir nuestro contenido. HTML como tal, no puede dar estilos, para eso usamos el lenguaje de hojas de estilo en cascada, mayormente conocido como CSS que se encargara de darle forma, estilos, entre otros.

Que es una etiqueta html?

Las etiquetas en html son las que nos ayudan a estructurar la información que tendrá nuestro sitio web, con ellas podremos hacer: encabezados, títulos, botones, poner imágenes, entre otros ... Las etiquetas html son las que le dicen al programa que es cada parte de lo que se muestra, es decir, las etiquetas html llevaran un orden en el que el programa leerá que es cada una de las estructuras.
Por ejemplo: Una página esta estructurada normalmente por header, main, section , div y footer.

Que es un atributo?

Los atributos en HTML son palabras clave que el desarrollador usa para agregar información y poder modificar con mayor facilidad sus etiquetas usando una hoja de estilos.

Atributos HTML
Atributo Uso
href="" Atributo usado en las etiquetas "a",
en este atributo se podrá poner la
dirección url de otro sitio web.
src="" alt="" Atributo usado en la etiqueta "img",
este consta de dos atributos, en el
atributo "src="" " se pondrá el nombre
(nombre de la imagen que este dentro
de una carpeta del proyecto que se esta
realizando ) o dirección url de un sitio
web. En el "alt="" " irá una referencia
de que trata la imagen

¿Que es un selector?

Un selector es la forma en la que un desarrollo web puede seleccionar un elemento especifico entre tantos tenga en su proyecto, también un selector le ayuda a un desarrollo a poder tener un mejor manejo de los estilos u cambios que desee hacer.
