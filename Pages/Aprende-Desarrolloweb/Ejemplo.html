Ejemplo de una pagina web

Hey!! Felicidades si has llegado hasta este punto, eso significa que te gusta aprender sobre el desarrollo web.

A continuación te dejare un ejemplo de como se vería estructurada una pagina web.

Ejemplo Html



< !DOCTYPE html>
< html lang="es">
< head>
....< meta charset="UTF-8">
....< meta name="viewport" content="width=device-width, initial-scale=1.0">
....< title>Nombre de la Página< /title>
....< link rel="stylesheet" href="styles.css">
< /head>
< body>
....< header>
........< h1>Encabezado de la Página< /h1>
........< nav>
............< ul>
................< li>< a href="#inicio">Inicio< /a>< /li>
................< li>< a href="#acerca">Acerca de< /a>< /li>
................< li>< a href="#contacto">Contacto< /a>< /li>
............< /ul>
........< /nav>
....< /header>

....< main>
........< section id="inicio">
............< h2>Sección de Inicio< /h2>
............< p>Bienvenido a mi página web.< /p>
........< /section>
........< section id="acerca">
............< h2>Acerca de< /h2>
............< p>Información sobre la página o el autor.< /p>
........< /section>
........< section id="contacto">
............< h2>Contacto< /h2>
............< p>Puedes contactarme a través de...< /p>
........< /section>
....< /main>

....< footer>
........< p>Derechos de autor © 2022 | Nombre de la Página< /p>
....< /footer>
< /body>
< /html>


Ejemplo Css



body {
....font-family: Arial, sans-serif;
....margin: 0;
....padding: 0;
....}

header{
....background-color: #333;
....color: #fff;
....padding: 10px;
....text-align: center;
....}

ul {
....list-style: none;
....padding: 0;
....}

ul li {
....display: inline;
....margin: 0 10px;
....}

main{
.... padding: 20px;
....}

footer{
....background-color: #333;
....color: #fff;
....text-align: center;
....padding: 10px;
....position: absolute;
....bottom: 0;
....width: 100%;
....}

Este es un pequeño código donde se implemento algunos de los elementos, con el fin de que te puedas dar una idea de como crear tu primer sitio web.
