Aprende CSS

Qué es CSS?

CSS, también conocido como hoja de estilos en cascada, este lenguaje es utilizado para poder darle color, formas e estilos, aprender css es un buen comienzo si estas interesado en el desarrollo web, al aprenderlo tendrás una gran variedad de animaciones y estilos sin limites para que los puedas implementar en tu página.

Como funciona Css?

Css funciona como el complemento de HTML, mientras html se encarga de la estructuración de un sitio web, css es el encargado de realizar los cambios visuales.

Uso de Selector

Los selectores tienen su propia forma de ser llamados en una hoja de estilos a como lo es css, como anteriormente se menciono, un selector es aquel con el que le daremos un nombre a una de nuestras etiquetas y poder seleccionarla sin afectar a las demás.

Los selectores "class" en la hoja de estilos son llamados por un punto, seguido del nombre asignado, al igual que los selectores "id" pero estos son llamados por un numeral (#).

Como usar los selectores?

HTML
< div class="div">< /div>


HTML
< div id="div">< /div>

CSS
.div {}


CSS
#div {}

Los tipos de selectores nos ayudaran a poder darle nuestros propios estilos a nuestro proyecto, dentro de ellos se pueden poner todo los estilos que se necesiten para lograr obtener el resultado que buscas.
